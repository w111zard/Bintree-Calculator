# Overview

This is simple calculator written by using binary tree. It supports brackets and operations '+' '-' '*' '/'


# Installation
```bash
$ make 
$ sudo make isntall
```

# Usage

```bash
$ bintree-calculator "30 + (4 * 5 - (2 * 6 + 12)) - 10"
$ 30 + (4 * 5 - (2 * 6 + 12)) - 10 = 16.000000
```
# Uninstall
```bash
$ sudo make uninstall
```
